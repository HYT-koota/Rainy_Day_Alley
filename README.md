# 雨天走巷子

竖屏优先的霓虹雨巷游戏 UI demo。玩家通过键盘或触控控制伞的高度、倾角和抬脚时机，躲避巷子里的行人、电动车、水坑、湿墙窄道、车辆、飞行/爬行蟑螂和非洲大蜗牛。

## 本地运行

```bash
npm install
npm run dev
```

打开 `http://localhost:3000`。

## 验证

```bash
npm run lint
npm run build
```

## 操作

- `W` / `↑`：升伞
- `S` / `↓`：降伞
- `A` / `D`：调整伞倾角
- `Space`：抬脚

移动端可以直接使用画面底部的触控按钮。
